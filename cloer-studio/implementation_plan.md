# Plan stworzenia strony dla Cloer Studio 🚀

Ten dokument zawiera krok po kroku plan budowy nowoczesnej, estetycznej strony internetowej dla content creatora, opartej na przesłanych przez Ciebie grafikach. Strona będzie zoptymalizowana pod kątem prezentacji rolek, postów i grafik.

## 🎨 System Projektowy (na podstawie zdjęć)
*   **Kolorystyka:** Sage Green (#98A89A - główny), Ciemny Grafit (#4A5D53), Biel/Off-white.
*   **Typografia:** Dekoracyjny font szeryfowy (podobny do *Cooper Black* lub *Recoleta*) dla nagłówków, czysty font bezszeryfowy (np. *Inter* lub *Montserrat*) dla treści.
*   **Styl:** Minimalizm, miękkie cienie, dużo "white space", estetyka "clean girl / studio look".

---

## 🛠 Krok 1: Inicjalizacja projektu i GitHub
1.  Utworzenie folderu projektu `cloer-studio`.
2.  Inicjalizacja projektu przy użyciu Vite (szybkość i nowoczesność).
3.  Utworzenie repozytorium na GitHub:
    *   `git init`
    *   `git add .`
    *   `git commit -m "Initial commit"`
    *   Połączenie z Twoim kontem GitHub.

## 🏗 Krok 2: Struktura strony (Sekcje)
Zgodnie z Twoimi wytycznymi i zdjęciami, zaimplementujemy:
1.  **Hero Section:** Nagłówek "Chcesz działać regularnie...", logo Cloer Studio i przycisk "Learn More".
2.  **O mnie:** Zdjęcie profilowe + opis Twojej misji.
3.  **Portfolio (Przykładowe prace):** Dynamiczna siatka z podziałem na Rolki (pionowe) i Posty.
4.  **Usługi:** Trzy kolumny (Tworzenie contentu, Wsparcie social media, Copywriting).
5.  **Czego używam?:** Sekcja z ikonami narzędzi (Canva, Notion, Instagram).
6.  **FAQ:** Rozwijana lista najczęstszych pytań (np. o terminy, sposób współpracy).
7.  **Kontakt:** Estetyczny formularz kontaktowy.
8.  **CTA & Popout:**
    *   Stały przycisk "Zacznijmy współpracę" w rogu.
    *   **Popout:** Okienko typu "Lead Magnet" lub szybki kontakt wyskakujące po 30 sekundach.

## 🖼 Krok 3: Dodawanie Twoich mediów (Checklista)
Zanim przejdziemy do finalizacji, będę potrzebować od Ciebie plików.
**Proszę, wrzuć mi tutaj (lub przygotuj do wklejenia) następujące elementy:**

- [ ] **Logo:** Cloer Studio (najlepiej w formacie .png bez tła lub .svg).
- [ ] **Zdjęcie Hero:** To z laptopem i rośliną lub inne główne tło.
- [ ] **Zdjęcie "O mnie":** Twoje profesjonalne zdjęcie.
- [ ] **Prace - Rolki:** 3-4 krótkie nagrania mp4 lub screeny z okładkami.
- [ ] **Prace - Posty:** 3-4 grafiki kwadratowe.
- [ ] **Ikony narzędzi:** Jeśli masz specyficzne, jeśli nie - użyję standardowych (Canva, Notion itp.).

## 🚀 Krok 4: Wdrożenie na Vercel
1.  Zalogowanie do [Vercel.com](https://vercel.com).
2.  Import repozytorium z GitHub.
3.  Konfiguracja domeny (Vercel automatycznie wygeneruje adres `cloer-studio.vercel.app`).
4.  **Automatyzacja:** Każda zmiana, którą zatwierdzimy, automatycznie odświeży stronę na serwerze.

## ✅ Co robimy najpierw?
1. Potwierdź, czy plan sekcji Ci odpowiada.
2. Gdy będziesz gotowa, zacznij podsyłać zdjęcia z checklisty w Kroku 3.
3. Ja w międzyczasie przygotuję bazowy kod struktury i stylu (CSS).
